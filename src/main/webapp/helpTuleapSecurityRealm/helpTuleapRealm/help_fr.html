En utilisant ce domaine, les utilisateurs Jenkins peuvent utiliser leur compte Tuleap pour se connecter au serveur Jenkins.
Pour pouvoir être authentifié en utilisant Tuleap, vous devez rajouter l'URL de redirection du serveur Jenkins dans votre application Tuleap OAuth2
L'URL de redirection est de la forme suivante: https://your_jenkins_server.example.com/securityRealm/finishLogin

REMARQUES :
  • Le module OAuth2 est un module Tuleap Enterprise
  • Votre serveur Jenkins DOIT être en HTTPS
  • L'usage du PKCE DOIT être activé dans votre application OAuth2 Tuleap
