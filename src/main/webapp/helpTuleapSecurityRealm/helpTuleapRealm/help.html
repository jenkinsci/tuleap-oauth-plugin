By using this realm, user will be able to use their Tuleap account to be connected on the Jenkins server.
To be authenticated with Tuleap, you have to add the Jenkins redirection endpoint in your Tuleap OAuth2 app.
The format of the Jenkins redirection endpoint is: https://your_jenkins_server/securityRealm/finishLogin

NOTES:
  • The OAuth2 module is a Tuleap Enterprise module
  • Your Jenkins server MUST be in HTTPS
  • The PKCE usage MUST be enabled in your Tuleap OAuth2 app
